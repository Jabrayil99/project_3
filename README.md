# Jabrayil Goyushov - Personal Portfolio

This is my personal portfolio website created for SITE 1101 course.

## Technologies Used
- HTML5
- CSS3
- Jekyll (for static site generation)

## Local Development
1. Install Jekyll: `gem install jekyll bundler`
2. Install dependencies: `bundle install`
3. Run locally: `bundle exec jekyll serve`
4. Open http://localhost:4000/site1101/

## Deployment
This site is hosted on GitHub Pages
